# 배경
golang 의 grom 을 사용해서 데이터 생성 날짜를 현재 시간으로 자동으로 insert 했다.
golang 코드에서 는  `time.Time` 을 사용해서 db 에 insert 한다

```
type testEntity struct {
	ID        string `gorm:"primarykey;type:varchar(10)"`
	...
	CreatedAt time.Time
	UpdatedAt time.Time
}
```
