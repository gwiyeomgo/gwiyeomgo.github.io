EKS와 ECR은 AWS에서 제공하는 별도 관리형 서비스
쿠버네티스 클러스터. Jenkins가 배포하는 Pod가 실행됨

```
[EKS 클러스터 (staging/production)]
       ↓
[ Golang 백엔드 앱 Pod  ]
       ↓
[ Service (ClusterIP or LoadBalancer) ]
       ↓
[ RDS - MySQL ]

```

* Pod(s) :ECR 이미지로 구성된 앱 컨테이너
